List of URLs

/dailyTotalStats

List the total number of cases (minimum and maximum) for all the days that have been logged so far, one entry per day. Adding /today to that route will return the total number of cases (minimum and maximum) for the current day, if it exists.

/dailyAgentStats

List the total number of cases (minimum and maximum) for all agents and all the days that have been logged so far, one entry per day per agent. Adding /[AgentName]/today to that route will return the total number of cases (minimum and maximum) for a specific agent and for the current day, if it exists.

/dailyAccountStats

List the total number of cases (minimum and maximum) for all agents and all the days that have been logged so far, one entry per day per agent. Adding /[AccountName]/today to that route will return the total number of cases (minimum and maximum) for a specific agent and for the current day, if it exists.
